What's in the ZIP?
  • config.h - Pre-configured with your WiFi, device ID, and cryptographic keys
  • ESP8266_C3DS_Sensor.ino - Main Arduino sketch
  • network.h/cpp - WiFi and HTTP handling
  • crypto.h/cpp - ECDSA signature generation
  • messaging.h/cpp - Message formatting
  • hardware.h/cpp - LED and button handling
Setup Instructions
  1. Extract the ZIP file
  2. Open ESP8266_C3DS_Sensor.ino in Arduino IDE
  3. Install required libraries: micro-ecc, ArduinoJson
  4. Select your board (NodeMCU 1.0 / Wemos D1 Mini)
  5. Upload to your device
